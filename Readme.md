# C++ Chat Application (Client–Server)

A simple client-server chat application built in C++ using TCP sockets and basic multithreading.

---

## Features

- Multi-client communication over TCP/IP
- Username-based messaging
- Timestamped messages
- Basic multithreading for handling multiple clients

---

## Tech Stack

- C++
- POSIX Sockets (TCP/IP)
- Multithreading (`std::thread`)
